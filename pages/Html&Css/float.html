# 元素设置左浮动却向右浮动

# 1. 浮动向左的元素第二行第一个dom往右边浮动问题

在某些情况下，在一个div盒子内所有元素向左浮动，当超出两行时，第二行第一个元素会向右边浮动。

这是因为 盒子高度不统一或未设置问题，可以使用给每个子元素增加 `min-height` 解决
